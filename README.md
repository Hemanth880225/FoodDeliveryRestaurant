# FoodDeliveryRestaurant

## Description

FoodDeliveryRestaurant is a simple restaurant food delivery web application built using Flask.
Users can browse menu items, view food images, and place orders through a web interface.

---

## Tech Stack

### Backend

* Python
* Flask

### Frontend

* HTML
* CSS
* Jinja2 Templates
* Bootstrap

### Database

* SQLite
* SQLAlchemy

### Other Tools

* JSON (for menu data)
* Git
* GitHub

---

## Features

* Browse restaurant menu
* View food items with images
* Simple order placement interface
* Clean and responsive UI

---

## Installation

Clone the repository

```
git clone https://github.com/Hemanth880225/FoodDeliveryRestaurant.git
```

Move into the project folder

```
cd FoodDeliveryRestaurant
```

Create virtual environment

```
python -m venv .venv
```

Activate virtual environment

Windows:

```
.venv\Scripts\activate
```

Install dependencies

```
pip install -r requirements.txt
```

Run the application

```
python run.py
```

Open in browser

```
http://127.0.0.1:5000
```
